This is a [Vite](https://vitejs.dev/) project bootstrapped with [`create-vite`](https://vitejs.dev/guide/#scaffolding-your-first-project).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`vite/font`](https://vitejs.dev/guide/features.html#fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
